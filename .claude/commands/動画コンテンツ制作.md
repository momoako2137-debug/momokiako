## STEP 2｜スライド構成案を作る

- 1スライド1テーマで構成案を作成
- 各スライドに：タイトル・本文（2〜3行）・画像提案（日本語）を入れる

- モモさんに確認してもらい、OKが出たら次へ

## STEP 3｜トークスクリプトを作る

- STEP 2の構成案と1対1で対応するスクリプトを作成
- 形式：「スライド1：（話す内容）」のようにスライド番号と紐付ける
- トーンは第0回台本（`main_courses/01_self_care/第0回_ようこそ動画_スクリプト.txt`）に合わせる
- ファイルに保存：`main_courses/01_self_care/slides/第○回_動画○_トークスクリプト.txt`
- モモさんに確認してもらい、OKが出たら次へ

## STEP 5｜スピーカーノート自動入力コード（GAS）を出す

STEP 3のトークスクリプトをもとに、以下の形式でGoogle Apps Scriptを出力する：

```javascript
function addSpeakerNotes() {
  var presentation = SlidesApp.getActivePresentation();
  var slides = presentation.getSlides();

  var notes = [
    // スライド1
    "（スライド1のトークスクリプト）",
    // スライド2
    "（スライド2のトークスクリプト）",
    // …全スライド分
  ];

  for (var i = 0; i < notes.length; i++) {
    if (i < slides.length) {
      slides[i].getNotesPage().getSpeakerNotesShape().getText().setText(notes[i]);
    }
  }

  Logger.log('スピーカーノートを' + notes.length + '枚のスライドに入力しました');
}
```

モモさんへの案内：
「GoogleスライドでPDFを開いたあと、拡張機能 → Apps Script にこのコードを貼り付けて▶実行してください。初回だけ権限の確認が出ますが許可すればOKです。」
