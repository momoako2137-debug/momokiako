【Claude Code を開くとき】

Claude Code＝作業用の別の会話ウィンドウ（黒い画面＝ターミナルから起動）です。

① Cursor で「ターミナル」を開く
② 次の2行を、そのままコピーして黒い画面に貼る
③ 貼るとき「2行貼りますか？」と出たら、中身を見て大丈夫なら「Paste」

cd "c:\Users\Owner\Documents\claude_workspace\momokiako"
claude

※ うまくいった日や、変な表示が出たときは、
　「05_メモ」のファイルに一言書いておくと次が楽です。

【続きから始めるとき】
　作業を終える前に「hikitsugi.txt」（ひきつぎ）にメモを書いておく。
　次の日は、先に  .\続き.ps1  を実行するか、続き.bat をダブルクリック。
　表示された「Claude に送る文」をコピーして、claude を開いたあと送る。
