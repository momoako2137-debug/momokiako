================================================================================
【付箋に貼る用】このファイルを開いて、下の「終了時」「再開時」を
　それぞれコピーして、付箋アプリに2枚貼っておくと便利です。
================================================================================


──────── コピー開始：終了時（作業をやめる直前）────────

【終了時】やること

① いちばん上のフォルダ（momokiako）で「hikitsugi」を開く
　（ファイル名 hikitsugi.txt＝ひきつぎ、と覚える）

② 次の2か所に、短くでOKなので書く
　・いま終わったところ（日付＋一行）
　・次にやること（一行でOK）

③ （よければ）Cursor か Claude Code に言う
　「hikitsugi.txt を、終わったことと次にやることに合わせて更新して」
　（口頭なら「ひきつぎを、終わったことと次に合わせて更新して」でもOK）

④ ファイルを保存して、パソコンを閉じる

──────── コピー終了：終了時 ────────


──────── コピー開始：再開時（続きを始めるとき）────────

【再開時】やること

① いちばん上のフォルダ（momokiako）で
　「続き.bat」をダブルクリックする
　※ Cursor を使うときは、ターミナルで次を打って Enter
　　.\続き.ps1

② 黒い画面にメモが出る。あわせて、送る文がコピーされる
　（「Copied to clipboard」と出たらOK）

③ ターミナルで次を打って Enter（まだなら）
　cd "c:\Users\Owner\Documents\claude_workspace\momokiako"

④ 次を打って Enter
　claude

⑤ Claude Code が開いたら、Ctrl+V で貼り付けて Enter
　（貼れないときは「Claude再開_コピペ一行」を開いてコピー）

⑥ 「hikitsugi.txt」を読んだうえで、続きを一緒に進める

──────── コピー終了：再開時 ────────


【補足】
・道に迷ったら「00_ももさん用」の「01_まずここ」を開く
・付箋は文字が小さくなるので、長すぎる文は削って自分用に短くしてOK
