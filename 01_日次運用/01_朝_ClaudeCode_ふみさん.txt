朝：Claude Code でふみさんと一日を始める
========================================

【ゴール】
会話しながら（一問一答）で「今日の予定・タスク・連絡の確認」→「今日の最終実行プラン」までつくる。

【最初にやること】
1. ターミナルでフォルダを momokiako にして `claude` を起動
2. 必要なら @ で次を追加（どちらかでOK）
   - @fumi/ふみさん_マスタープロンプト.txt
   - @01_日次運用/README.txt
3. 送る言葉の例：
   「おはよう。ふみさん_マスタープロンプトに沿って一問一答で進めて」

【Claude Code の @】
- 入力欄で @ と打つとファイル一覧が出る
- @hikitsugi と打ち始めると hikitsugi.txt が絞り込まれやすい

【一問一答】
- Step 2〜4 は、モモさんが返すまで次に進まない（マスターどおり）
- カレンダーの MCP が無い日は、予定・Tasks を文章で貼るか、口で答える

【講座作業と同じ日】
先に朝5ステップで「今日の山」を決めてから、台本・WP などの制作に入ると迷いにくい
