fumi/ … ふみさん（朝・夜運用）の置き場
========================================

【いちばん最初に迷ったら】
→ **`00_入口/README.txt`**（リポジトリ全体の地図）
→ 毎日の細かい地図は **`01_日次運用/README.txt`**

【何を置くか】
- ふみさん_マスタープロンプト.txt … 役割・ミッション・朝5ステップ・夜の振り返り・コミュニケーションルール（原本）
- 必要に応じて増やす例:
  - 朝テンプレ短縮版.txt（忙しい日用）
  - MCP_Calendar_Tasks_メモ.txt（接続手順・制限の一言メモ）

【Cursor での使い方】
1. 新しいチャットで朝ルーティンを始めるとき、
   「おはよう。fumi/ふみさん_マスタープロンプト.txt に沿って一問一答で」と書く
   （またはファイルを @ で添付）
2. 講座コンテンツ制作とは別枠の「ふみさん」モードだとブレにくい

【このリポジトリでの現実的な線引き】
- Google カレンダー／Tasks: MCP が繋がっていれば取得。未設定ならモモさんが予定・タスクを貼るか要約で渡す
- NotebookLM: 自動取得はしない。マスターの Step1 の「入れましたか？」は手動確認のまま
- Notion・メール・LINE・SNSのDM: エージェントが勝手に開けない。確認した結果をモモさんから共有

【フォルダの役割分け】
- 00_入口/ … リポジトリ全体の迷子用・ひとつの入口（地図）
- 01_日次運用/ … 毎日のナビ（朝・夜・MCP／手動）の詳細入口
- 00_ももさん用/ … 講座・事業まわりの手順・地図（番号付き）
- fumi/ … ふみさん用プロンプトの原本（朝5ステップ・夜）
- hikitsugi.txt … セッション切れ目の引き継ぎ（講座タスク＋必要なら技術メモ）
